
## Final acceptance checklist

- **Design tokens only:** All colors, spacing, fonts, shadows, and gradients derive from :root tokens; no ad-hoc hex or inline styles.
- **Grid + Flex:** Layouts use CSS Grid for sections and Flex for alignment; responsive breakpoints implemented.
- **Multipage nav:** Header links correctly route to Home, About, Case Studies, Contact with aria-current on each page.
- **CTAs:** Contact and WhatsApp CV buttons present in header, hero (Home), and footer across pages.
- **Accessibility:** Visible focus states, alt text, labels and roles set; keyboard-friendly; reduced motion respected.
- **Imagery:** Strategic, tasteful placement; professional portrait; grayscale affiliations; justice iconography used sparingly.
- **Performance & SEO:** Lazy-loaded images, fast fonts, minimal JS, complete meta and schema.
- **Copy & compliance:** Professional tone, clear disclaimers, confidentiality respected.
- **Testing:** Verified on mobile, tablet, desktop; cross-browser checked (Chrome, Firefox, Edge, Safari).
